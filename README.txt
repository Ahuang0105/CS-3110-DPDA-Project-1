The IDE used to build this programs is Eclipse, the rar file "CS 3110 Program project.rar" file
contain 1 folder call "PDA". Please place the whole folder under your "eclipse- workspace" folder.
Next open your Eclipse software and selete "File" at the very top left corner, next click "open 
Projects from file System". Under the "import source" please selet the path to "PDA" folder and 
clip "finish". Also, the source code is locate in the folder call "bin".
